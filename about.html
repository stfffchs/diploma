Diplomarbeit Kontext

  • Responsive

    Die Website soll für Desktop und Mobile optimiert sein und wird als Link via GitHub Issue abgegeben.

  • Aufbau

    Die Arbeit soll nich künstlerisch, sondern grafischer ud dienender Natur sein.

  • Unabhängig

    Der gesamte Inhalt muss so aufgebaut sein, dass keine externen Dienste in Anspruch genommen werden.

  • State of the Art

    Die Arbeit soll korrekt und mit den üblichen Herangehensweisen aufgebaut werden.

  • User-Experience

    Das Konsumieren der Arbeit dauert mindestens 15 Minuten.

  • Medientypen

    Es müssen mindestens vier Medientypen verdendet werden.
